# Aplicação de Design Patterns (GoF) em C# WPF

Repositório destinado à Situação de Aprendizagem da disciplina de Desenvolvimento de Sistemas. O objetivo deste projeto é demonstrar a aplicação prática e a justificativa arquitetural de um Padrão de Projeto GoF em uma aplicação desktop.

## Padrão Escolhido: Facade (Fachada)
O padrão **Facade** (Estrutural) foi implementado para resolver problemas de alto acoplamento em sistemas complexos. Ele fornece uma interface unificada e simplificada para um conjunto de interfaces (subsistemas) de regras de negócio, separando completamente a interface gráfica (WPF) das lógicas de back-end.

##  Estrutura do Repositório
* `/docs`: Contém o Relatório Técnico detalhado (PDF) abordando a fundamentação teórica, análise crítica, comparação de abordagens e aplicações reais do padrão no mercado.
* `/(Em breve)`: Código-fonte do protótipo funcional desenvolvido em C# e WPF.

##  Tecnologias Utilizadas
* C# (.NET)
* Windows Presentation Foundation (WPF) para a Interface Gráfica
* IDE: JetBrains Rider
* Padrão Arquitetural: Facade (GoF)
